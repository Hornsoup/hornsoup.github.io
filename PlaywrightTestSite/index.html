CI/CD Workflow

GitHub Actions auto-runs Playwright tests and deploys if passing: 
name: Test & Deploy
on: [push]
jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: microsoft/playwright-action@v1
        with:
          runTests: true
  deploy:
    needs: test
    runs-on: ubuntu-latest
    steps:
      - run: echo "Deploy to hosting platform"
Site last tested: [timestamp] | Status: ✅ passing
